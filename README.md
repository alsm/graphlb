# graphlb
graphlb is a library which contains all the graph datastructures and algorithms

# Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  delimiter_tree:
    github: mettuaditya/graphlb
```
Then run `crystal deps`
